Wellness Tracker Client (Static)

API Base:

This client calls: Get All, Get By ID, and a subset endpoint for each table.

Users

Endpoints: /api/users, /api/users/{id}

Habits

Endpoints: /api/habits, /api/habits/{id}, /api/users/{userId}/habits

Daily Logs

Endpoints: /api/dailyLogs, /api/dailyLogs/{id}, /api/users/{userId}/dailyLogs

Habit Entries

Endpoints: /api/habitEntries, /api/habitEntries/{id}, /api/dailyLogs/{dailyLogId}/habitEntries
